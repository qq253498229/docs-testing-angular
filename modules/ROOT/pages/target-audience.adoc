= 目标受众

学习目标

- 阅读本书时需要了解的 Angular 主题
- 了解 Angular 核心概念的学习资源
- 选择您感兴趣的章节

本指南的目标受众是中级 Angular 开发者。您应该熟悉 Angular 的核心概念。

TIP: 测试，而不是实现

本指南教您如何测试 Angular 应用程序的组件和服务等部分。它假设您知道如何实现它们，但不知道如何进行正确的测试。如果您对 Angular 的核心概念有疑问，请参考 https://angular.io/docs[官方的 Angular 文档]。

如果您尚未使用过某些单独的概念，例如指令，那也没有关系。您可以简单地跳过相关章节，选择您感兴趣的章节进行阅读。

TIP: JavaScript 和 TypeScript 熟练度

此外，本指南不是 JavaScript 或 TypeScript 的入门介绍。它假设您具备足够的 JavaScript 和 TypeScript 知识，可以编写所需的实现和测试代码。当然，本指南将解释常用于测试的特殊惯用法。

官方的 Angular 文档提供了一份全面的 https://angular.io/guide/testing[测试指南]。这是推荐阅读，但本指南并不假设您已经阅读过它。
