= 术语

学习目标

- 本书中使用的术语
- 符号约定

在我们深入讨论之前，对技术术语进行一个简短的说明。

在 Angular 的上下文中，某些词汇具有特殊的含义。在更广泛的 JavaScript 上下文中，它们可能有其他许多含义。本指南试图通过使用不同的大小写来区分这些含义。

当提到核心 Angular 概念时，本指南使用 *大写字母*：

_模块（Module）、组件（Component）、服务（Service）、输入（Input）、输出（Output）、指令（Directive）、管道（Pipe）_ 等。

当在一般意义上使用这些术语时，本指南使用 *小写字母*：

_模块（module）、组件（component）、服务（service）、输入（input）、输出（output）_ 等。
